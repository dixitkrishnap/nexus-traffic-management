##  Features

-  Intelligent Signal Optimization  
-  Real-time Traffic Analytics  
-  Emergency Vehicle Priority  
-  AI-based Decision System  
-  REST API + WebSocket Support  

---

##  How It Works

Traffic data is analyzed in real-time using AI to dynamically adjust signals, reduce congestion, and provide insights through a live dashboard.

- Frontend: HTML, CSS, JavaScript  
- Backend: FastAPI (Python)  
- Database: SQLite  
- Deployment: Vercel + Render
